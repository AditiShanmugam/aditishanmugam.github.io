Superpixel Masking and Image Inpainting with Multi Exposure Fusion

Anomaly Detection and Correction using Supermixel Masking and Inpainting using a mask based curriculum learning and multi image exposure. Two variants of the network, GAN and Autoencoder based architectures were developed. I worked with Aditya Kumar Pal a former M.Tech student at the Indian Institute of Science, during my Internship at the Visual Computing Lab. The code for the project is yet to be released.

Code

Fire and Smoke Detection using Computer Vision

This work is to develop a simple Computer Vision-Based Fire Alarm System. This work is heavily inspired by FireNet. Three models were trained and tested using MATLAB and deployed on a RaspberryPi Model 4B. The models were finetuned to make predictions with high confidence on low quality CCTV images. The main objective behind the project was to develop an inexpensive and efficient fire detection system.

Code

Source Free - Multi Label Domain Adaptation

This work introduces a novel concept of Source-Free Multi-Label Domain Adaptation (SF-MLDA) using graph convolution networks (GCN) and Co-teaching based methods. This is an ongoing project at the Visual Computing Lab as part of my internship with the Indian Institute of Science, Bangalore. I work under Vikash Kumar, PhD candidate at the institute.

Multi-Modal Machine Learning for Object Detection.

The main aim of this project is to improve object detection using different modalities (Here, Image-Text pairs) of data for Zero-shot or Few-shot learning. This work is currently underway for my Senior year capstone project.
